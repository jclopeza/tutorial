Un SW con el que podrás olvidar para siempre el tedioso cálculo de las congruencias.

Algo de teoría: congruencias
Dados 3 números enteros, a, b y c, diremos que "a es congruente con b módulo c si la diferencia entre a y c es un múltiplo de b. Dicho de otro modo, el resto de dividir a entre b es c."

a es congruente con b módulo c si existe un entero k tal que a-c=kb

Ejemplos


Veamos algunos ejemplos para aclarar el concepto
  • ¿10 es congruente con 3 módulo 4?, sí, porque 10-4 es un múltiplo de 3
  • ¿11 es congruente con 4 módulo 3?, sí, porque 11-3 es un múltiplo de 4
  • ¿11 es congruente con 4 módulo 6?, NO, porque 11-6 NO es un múltiplo de 4
